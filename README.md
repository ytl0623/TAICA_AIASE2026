# 生成式 AI 應用系統與工程 (AIASE 2026)

**Generative AI Application Systems and Engineering**

> 國立成功大學 資訊工程研究所 ｜ 114 學年度 第 2 學期 ｜ 3 學分
> 授課教師：莊坤達 (ktchuang@mail.ncku.edu.tw)

---

## 課程簡介

本課程以「生成式 AI 應用系統」開發為核心，帶領學生從需求分析、系統設計到實作與部署，完成一個生成式 AI Web 服務。內容涵蓋軟體工程 SDLC、前後端技術、資料流程與工作流、CI/CD 與 MLOps/LLMOps、AWS 雲端環境、分散式 AI Infra、LLM 微調與評測、Agent 工作流與 MCP/ADK/Agent-to-Agent 架構、Token 經濟學、Prompt 優化、多輪對話設計、幻覺減少與 LLM 資安議題。
